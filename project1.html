Insanely Bouncy Castle

Welcome to Insanely Bouncy Castle! I created this short game in 48 hours for St. Lawrence College's 2024 SLC game jam. Despite not having a team, I was able to place third in the bouncy castle theme.
I'm quite proud of what I was able to achieve because this was one of the first real, semi-proper games I've ever made despite only recently beginning my game development journey. All of the assets besides audio were created by me.

About this game

I was inspired to create this game by two distinct games, Jumpking and Getting Over It. I attempted to combine those two games with bouncy castles, and this is what I ended up creating.
It was interesting to try to create a very tall bouncy castle because, like those two games, I wanted the map to be all together and have the potential to lose progress if you ball or make a bad jump. This would require a more vertical map.
