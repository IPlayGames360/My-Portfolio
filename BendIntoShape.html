Bend Into Shape

Bend Into Shape was my second attempt at making a game. I developed it solo in just 48 hours for the 2025 SLC Game Jam, where the theme was "Bent Out of Shape". I'm proud to say it earned 2nd place. I'm really happy with how the game turned out — especially how it feels to play. This was also the first 3D game I’ve ever developed. All of the assets (aside from audio) were created by me from scratch.

About this game

Bend Into Shape was inspired by Snakebird and the TV show Hole in the Wall. I thought the worm-like movement from games like Snakebird and Snake, combined with
the "fit the shape" concept from Hole in the Wall, could make for an interesting and fun challenge.
The game is played on a 17x17 grid, where the player controls a worm and must maneuver into the correct shape to pass through cutouts. Each successful pass adds another body segment, increasing your length and making the game progressively harder. To balance this, each cutout approaches more slowly, giving you more
time to adjust. The maximum score is 289 — good luck to anyone who can make it that far!
The most challenging part of this project for me was creating the cutout spawning system and managing the movement of each player segment. If I had more time, I would’ve liked to improve the movement mechanics — maybe adding a backup or undo function — and polish the visuals and audio.
I might return to this project one day. It feels like it could become a nice, polished little game with some extra love.
